Amor de visita


é livre,
é leve,
é livro.

é louvor,
é L'amour,
é lamento.

é ira,
é hora,
é reza.

é literário,
é gregário,
é necessário.

é técnica,
é espontâneo,
é métrica.

é opinião
é incômodo,
é besteira.

é difícil de entender,
é simples de entender,
é baseado no querer.

é curto,
é longo,
é lógico.

é pouco,
é muito,
é fantástico.

é tudo aquilo que um dia quis ser,
mas era perda de tempo, deixado pra lá.

é encanto,
marginalizado ao canto,
escanteio.

é lírico,
é crítico,
é sem freio.

é errado,
porque palavras não são medidas,
brincamos com semântica e tudo.
Se o maior pecado é o que sai da boca, o meu tem verso,
rima,
e conteúdo.

-Mpz
